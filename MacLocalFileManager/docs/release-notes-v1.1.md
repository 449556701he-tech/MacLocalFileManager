# MacLocalFileManager v1.1.0

v1.1.0 把 beta1.0 的 Spotlight 风格搜索窗补成更像日常 macOS 工具的入口：常驻菜单栏、全局快捷键、关闭后后台保留，少一点折腾，多一点随手可用。

## 本版重点

- 菜单栏常驻搜索图标，点击后打开快捷面板。
- 快捷面板支持显示/隐藏搜索、打开扫描设置、刷新文件索引和退出程序。
- 默认全局快捷键 `⌃F`，用于快速唤起或隐藏搜索窗口。
- 关闭主窗口后程序继续在后台运行，可通过菜单栏或快捷键再次唤起。
- 快捷面板改为白底黑字、macOS 风格圆角，复杂桌面背景下更清楚。
- 快捷面板使用自绘 Qt 小窗，不使用 Qt 原生托盘菜单，避开当前 macOS 上的原生菜单崩溃问题。
- 全局快捷键优先使用 Carbon 注册；失败时降级尝试 AppKit 监听，并在界面显示状态。

## 修补更正

- 修正点击菜单栏图标打开菜单时可能崩溃的问题。
- 修正 `⌘⌥Space` 与系统搜索冲突、`⌃⌥Space` 与输入法切换冲突后的快捷键方案。
- 修正菜单栏快捷面板不跟随菜单栏图标位置的问题。
- 修正快捷面板透明度过高、深色桌面背景下文字不清楚的问题。
- 修正快捷面板阴影影响文字渲染的问题。
- 修正快捷面板圆角外仍露出直角窗口底色的问题。
- 设置页保留全局快捷键开关，关闭后下次启动生效。

## 仍然不做的事

- 不接入在线 AI。
- 不内置重量级图片模型。
- 不做音频、视频识别。
- 不删除用户文件，只维护本地 SQLite 索引。
