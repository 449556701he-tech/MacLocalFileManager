# MacLocalFileManager beta1.0

beta1.0 是第一个偏日常可用的公开测试版，重点从“能搜”转向“像 macOS 工具一样顺手”。

## 本版重点

- 类 Spotlight 的无边框半透明窗口。
- 折叠态默认出现在屏幕上方黄金分割阅读位，输入后展开并居中。
- 支持文件名、路径、Office/PDF 正文、图片 OCR、图片标签和相似图片检索。
- 图片结果优先显示缩略图，文档结果显示路径、时间、大小和内容摘要。
- 图纸 / CAD / 清单筛选默认隐藏，可在设置中打开。
- 搜索筛选栏和结果区提高不透明度，复杂背景下更易读。
- 图片识别优先使用 Apple Vision 系统能力；没有可选依赖时会降级到离线基础索引。

## 修补更正

- 修正输入文字后窗口展开与搜索框宽度不一致的问题。
- 修正无边框窗口无法拖动的问题。
- 调整窗口外缘，去掉直角外框，贴近 macOS 聚焦搜索的圆角玻璃风格。
- 设置入口补回工程类筛选开关。
- 说明文档移除外部模型接口相关内容，避免把未内置的模型能力误写成当前功能。

## 下一步

- 全局快捷键快速唤起。
- 菜单栏常驻入口。
- 更无感的后台索引刷新。
- 更稳定的剪切板聚合搜索。
